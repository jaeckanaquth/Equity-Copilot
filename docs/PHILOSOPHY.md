# Philosophy — Intent & Guardrails

Why the system exists. What drift it prevents. **Status:** Final; project closed for current slice.

---

## Core Objective

**Better long-term, risk-adjusted returns through improved reasoning discipline.**

---

## The Problem

Human investors:

- Rewrite past beliefs
- Rationalize outcomes
- Forget assumptions
- Conflate outcome with correctness

This system exists to **prevent self-deception**.

---

## Design Philosophy

- **Facts immutable** — Snapshots fixed in time; no retroactive reinterpretation.
- **Beliefs immutable** — Claim text never changes; new thinking requires a new belief.
- **Evolution append-only** — Lifecycle events record change; no event mutates the past.
- **Decisions historical** — `revised` and `abandoned` are markers, not state transitions.
- **State derived, never stored** — Current decision, tension density, trajectories: computed from the log.
- **LLM assistive only** — Drafting, analysis, explanation; never creates beliefs or records decisions.

---

## Causal Chain

```
Clear belief → Structured review → Explicit decisions → Pattern awareness
       → Better capital allocation → Improved compounding
```

Returns are **downstream of reasoning**.

Never the primary optimization target.

---

## Guardrail

When in doubt, preserve epistemic integrity.
